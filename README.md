# FuzzyTEP Web

Веб-платформа для анализа технико-экономических показателей предприятия с использованием нечеткой модели.

## Что умеет приложение

- создавать, редактировать и удалять предприятия
- вводить показатели за период
- рассчитывать итоговую оценку предприятия через нечеткий анализ
- показывать числовой результат, словесную интерпретацию и пояснение
- хранить историю анализов в SQLite

## Запуск

1. Откройте терминал в папке проекта.
2. Выполните команду:

```bash
python3 app.py
```

3. Откройте в браузере:

```text
http://127.0.0.1:8000
```

## Технологии

- Python 3
- SQLite
- HTML, CSS, JavaScript

Приложение не требует установки сторонних библиотек.
